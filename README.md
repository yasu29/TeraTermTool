# SSH Auto Login Tool（TeraTermTool）

---

## 1. 概要
Tera Termを利用したSSH接続を効率化するGUIツールです。  
接続先IPの選択／入力、認証情報の入力、ログ確認を一つの画面で操作できます。

本ツールは、日常的に複数サーバへ接続する運用作業の効率化を目的としています。  
GUI化により、手入力やコマンド実行の手間を削減し、ヒューマンエラーの防止にも寄与します。

Tera Termマクロ（TTL）による接続制御を行っており、接続失敗時やタイムアウト時の簡易エラーハンドリングにも対応しています。

---

## 2. フォルダ構成

```

TeraTermTool/
├─ START_LOGIN.bat          # 起動用バッチ
├─ conf/
│  ├─ IpList.csv            # 接続先IP一覧（任意）
│  └─ DefaultAccount.csv    # 初期ユーザー／パスワード（任意）
├─ lib/
│  ├─ LoginGui.ps1          # GUI本体
│  └─ Main.ttl              # Tera Termマクロ
└─ log/                     # ログ出力先

```

---

## 3. 使用方法

1. `START_LOGIN.bat` を実行  
2. GUIが起動する  
3. IPアドレスを選択または入力  
4. ユーザー／パスワードを入力  
5. 「ログイン」ボタンを押下  

※IPリスト未設定の場合でも、手入力で接続可能です。

---

## 4. 動作環境

以下の環境で動作を確認済みです。

- OS：Windows 10 / 11  
- PowerShell：Version 5.1.22621.6060  
- Tera Term：Version 4.106 (SVN# 9298)

※上記以外の環境でも動作する可能性がありますが、未検証です。  

※Tera Termのインストールパスは以下を想定しています  
C:\Program Files (x86)\teraterm\  

※異なるパスにインストールしている場合は、`LoginGui.ps1`内のパスを環境に合わせて変更してください  

---

## 5. 免責・注意事項

- 本ツールは自己責任で使用してください  
- 環境によってはTera Termのパスを調整する必要があります  
- パスワードは現状平文で扱われるため、取り扱いには十分注意してください  
- ログ出力先フォルダ（log）が存在しない場合は、自動生成を試行します
- プロンプト判定：マクロは標準的なプロンプト記号（$, #, %）を待機します  
  接続後、一定時間内にプロンプトを検出できなかった場合はタイムアウトとして終了します
  カスタムプロンプトを使用している環境では、正常に動作しない場合があります  

---

## 6. 主な機能

- IPアドレスの選択／手入力（リスト＋直接入力対応）  
- ユーザー／パスワード入力  
- ワンクリックでSSH接続  
- ログフォルダの参照  
- ToolTipによる操作ガイド表示  
- IP形式バリデーション（TryParseによる検証）  
- 接続失敗時の簡易エラーハンドリング
- プロンプト待機タイムアウト検知

---

## 7. UI設計

- FlowLayoutPanel（縦方向）によるシンプルなレイアウト構成  
- GroupBoxで「接続情報」と「操作」を明確に分離  
- GroupBox内部にPanelを配置し、レイアウトの重なりを防止  
- AutoSizeを活用し、環境差異による表示崩れを抑制  

---

## 8. ログ仕様

- ログは `log` フォルダ配下に出力されます  
- ファイル名は「接続先IP + 日時」で生成されます  

例：
```

192.168.1.1_20260505-153000.log

```

※ログ出力はTTL側で制御しています  
※ログフォルダが存在しない場合は、TTL側で自動生成を試行します

---

## 9. 設定ファイル

### ■ IpList.csv（任意）
接続先IPの一覧を定義します。

```

192.168.1.1
192.168.1.2

```

※未作成でも手入力で利用可能です。

---

### ■ DefaultAccount.csv（任意）
初期表示するアカウントを設定します。

```

root,hogehoge

````

## 10. バリデーション

IPアドレスは以下のメソッドで検証しています。

```powershell
[System.Net.IPAddress]::TryParse(...)
````

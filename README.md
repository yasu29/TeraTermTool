# SSHポートログインツール 利用ガイド

## 1. 概要
本ツールは、PowerShell GUIで接続先を選択し、Tera Termを利用してSSH自動ログインおよび操作ログの自動取得を行うツールです。
特定のTera Termバージョンで発生する「マクロのハングアップ」を回避するため、本体プログラムが直接マクロ制御する安定志向の設計を採用しています。

## 2. フォルダ構成
```text
TeraTermTool/
├─ START_LOGIN.bat      # 実行用バッチファイル
├─ conf/                # 設定ファイル格納フォルダ
│  ├─ IpList.csv        # 接続先IPリスト
│  └─ DefaultAccount.csv # デフォルトのユーザー/パスワード
├─ lib/                 # プログラム本体フォルダ
│  ├─ LoginGui.ps1      # PowerShell GUIスクリプト
│  └─ Main.ttl          # Tera Termマクロ本体
└─ log/                 # ログ出力フォルダ（自動生成）
```

## 3. 使用方法
1. `conf/IpList.csv` に接続先IPアドレスを記載します。
2. `conf/DefaultAccount.csv` にユーザー名、パスワードを記載します。
3. `START_LOGIN.bat` をダブルクリックで起動します。
4. GUIで接続先を選択し「ログイン」を押すと、Tera Termが起動し自動操作が開始されます。

## 4. 動作環境
以下の環境で動作を確認済みです。
* **Tera Term**: Version 4.106 (SVN# 9298)
* **PowerShell**: Version 5.1.22621.6060
* **OS**: Windows 10 / 11

## 5. 免責・注意事項
* **動作保証**: 動作確認済み以外のバージョン（Tera Term 5.x系や古いPowerShell環境など）では、正常に動作しない可能性があります。
* **セキュリティ**: パスワードをCSVに保存する際は、端末の管理に十分注意してください。
* **プロンプト判定**: マクロは標準的なプロンプト記号（$, #, %）を待機します。サーバ側の設定でこれらが表示されない場合、マクロが停止することがあります。
